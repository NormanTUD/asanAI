\section{Dense Layer und ReLU}
% ============================================================

\begin{frame}{Baustein 1: Das Dense Layer}
\begin{columns}
\begin{column}{0.55\textwidth}
\textbf{Ein Neuron = eine einfache Rechnung:}
\vspace{0.2cm}

Eingabe: $x = 3$\\[0.2cm]
Gewicht: $w = 0.5$, Bias: $b = -1$\\[0.2cm]
$$y = w \cdot x + b = 0.5 \cdot 3 + (-1) = 0.5$$
\vspace{0.3cm}

Das ist nur eine \textbf{Gerade}!\\[0.2cm]
Egal wie viele Neuronen -- solange alles linear ist, bleibt es eine Gerade.
\end{column}
\begin{column}{0.45\textwidth}
\begin{tikzpicture}[scale=0.75,>=Stealth]
