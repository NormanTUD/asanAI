① Individual Sinusoidal Waves per Dimension

Each dimension of the positional embedding is a sine or cosine wave at a different frequency. Low dimensions oscillate fast (fine-grained position), high dimensions oscillate slowly (coarse position). Think of it like a clock: the second hand spins fast, the hour hand spins slowly — together they uniquely identify any time.

② The Circle Intuition

Each pair of dimensions (sin, cos at the same frequency) traces a circle. As position advances, the point rotates. Different frequency pairs rotate at different speeds. The highlighted position is shown as a bright dot.

③ Dot Product = Relative Distance

The dot product of two positional encodings depends only on their distance, not absolute position. This plot shows dot product of the highlighted position with every other position. Notice the symmetric, distance-dependent pattern.

④ Full Positional Encoding Heatmap

Each row is a position (token index), each column is an embedding dimension. Color encodes the value (blue = −1, black = 0, orange = +1). Notice the "barcode" pattern: low dimensions change rapidly, high dimensions change slowly — like binary counting but continuous.

⑤ Binary Counter Analogy

The top row shows actual binary representations. The bottom row shows the sinusoidal encoding. See how the sinusoidal version is a smooth, continuous generalization of binary counting!
