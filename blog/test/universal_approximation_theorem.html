Globale Parameter

Einzelne ReLU-Einheiten

Jede Einheit berechnet: aᵢ · ReLU(wᵢ · x + bᵢ)
wᵢ = Gewicht (Steigung), bᵢ = Bias (Knickposition), aᵢ = Ausgabegewicht

📐 Aktuelle Formel

f(x) = bias + Σ aᵢ · max(0, wᵢ·x + bᵢ)

💡 Erklärung

So funktioniert es:

1. Jede ReLU-Einheit erzeugt einen Knick an der Stelle x = -bᵢ/wᵢ

2. Links vom Knick ist der Beitrag 0 (abgeschnitten!)

3. Rechts vom Knick steigt/fällt die Linie mit Steigung aᵢ·wᵢ

4. Durch Addition aller Einheiten entsteht eine stückweise lineare Funktion

5. Mehr Einheiten = mehr Knicke = bessere Approximation! 🎯

🔬 Zerlegungsansicht – Wie sich die Approximation aufbaut

Diese Ansicht zeigt, wie jede ReLU-Einheit einzeln aussieht und wie sie sich Schritt für Schritt zur Gesamtapproximation aufsummieren. Von links nach rechts wird jeweils eine weitere Einheit hinzugefügt.
