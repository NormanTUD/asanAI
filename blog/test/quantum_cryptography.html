2. Shor's Algorithm — How Quantum Computers Break RSA

Shor's algorithm converts the factoring problem into a period-finding problem, which quantum computers solve exponentially faster using the Quantum Fourier Transform.

🎯 The Core Insight: Factoring → Period Finding

For N = p × q, pick random a < N. Compute a^x mod N for x = 0, 1, 2, 3... The result is periodic! Finding that period r lets us factor N.

📐 Lattice-Based Cryptography (CRYSTALS-Kyber / ML-KEM)

The hardest problem: Given a "noisy" point near a lattice, find the closest lattice point. This is the Learning With Errors (LWE) problem.

3 8

🌳 Hash-Based Signatures (SPHINCS+)

Security based purely on hash function properties. Uses Merkle trees — a tree of hashes where the root authenticates all leaves.

3

📡 Code-Based Cryptography (Classic McEliece)

Based on error-correcting codes. The hard problem: decoding a random linear code (finding the original message from a noisy codeword).

🧮 Multivariate Polynomial Cryptography

Based on solving systems of multivariate quadratic equations — proven NP-hard in general.
