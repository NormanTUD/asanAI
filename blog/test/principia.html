Principia Mathematica — Interactive Explorer

Russell & Whitehead (1910–1913) · Explore the foundations of mathematics through logic

1 Propositional Logic — The Bedrock

Principia builds all of mathematics from a tiny set of logical connectives. Use the sliders to set truth values for p and q, then see every connective evaluated in real time.

Why it matters: Russell & Whitehead showed that just ~ (not) and ∨ (or) are sufficient to define all propositional logic. Every other connective (∧, →, ↔) is defined in terms of these two.

2 Truth Tables — Tautologies & Contradictions

A key goal of Principia was to identify tautologies — formulas true under every assignment. Choose a formula to see its full truth table and whether it's a tautology.

3 Type Theory — Taming Russell's Paradox

The famous paradox: "Does the set of all sets that don't contain themselves contain itself?" Russell introduced a hierarchy of types so that a set at level n can only contain members of level n−1. Use the slider to explore different type levels.

The fix: "The set of all sets that don't contain themselves" is ill-formed in type theory — you can't ask whether a set of type n contains itself, because it can only contain objects of type n−1.

4 Building Numbers from Pure Logic

Perhaps the most famous achievement: defining natural numbers as classes of classes. 0 = the class of all empty classes. 1 = the class of all singleton classes. n+1 = the successor of n. Slide to see how numbers are constructed.

The famous proof: It takes Russell & Whitehead until page 379 of Volume I to prove that 1 + 1 = 2. The comment reads: "The above proposition is occasionally useful."

5 Modus Ponens — The Engine of Deduction

Principia uses very few inference rules. The most important is Modus Ponens: "If p is true, and p → q is true, then q is true." Build a chain of deductions by choosing premises.

6 Predicate Logic & Quantifiers

Beyond propositions, Principia uses ∀ (for all) and ∃ (there exists) to talk about properties of objects. Adjust the domain size and predicate to see how quantified statements evaluate.

7 The Logicist Programme — Big Picture

The grand claim: All of mathematics reduces to logic. Slide through the layers to see how each mathematical concept is built from the one below it.

Historical note: In 1931, Gödel showed that any consistent system powerful enough to express arithmetic must contain true statements it cannot prove — imposing fundamental limits on the Principia programme. But the work remains one of the greatest intellectual achievements in history.
