What is the Grundlagen?

Published in 1884, Die Grundlagen der Arithmetik is Frege's attempt to answer one deceptively simple question:

"What is the number one? What does the symbol '1' mean?"

Frege argued that previous answers fell into three traps:

  • Psychologism — treating numbers as mental images or ideas in our heads.
  • Empiricism (Mill) — treating numbers as properties of physical heaps of things.
  • Formalism — treating numbers as meaningless symbols manipulated by rules.

Frege's revolutionary alternative: Numbers are logical objects, defined purely through concepts and their extensions. Use the tabs above to explore each key idea interactively.

Idea 1: Anti-Psychologism

Frege's first move is negative: numbers are NOT mental images, feelings, or subjective ideas. They are objective, mind-independent entities.

"One must separate the psychological from the logical, the subjective from the objective."
— Frege, Grundlagen, Introduction

Psychologism: "The number 3 is a mental image I form when I think of three things."

Frege's Critique: If numbers were mental images, then your "3" and my "3" would be different (since we have different minds). But 2+1=3 is true for everyone. Mathematics would collapse into subjective psychology. We couldn't even disagree about arithmetic, because we'd each be talking about our own private ideas.

Idea 2: Concepts and Objects

Frege draws a sharp distinction between concepts (predicates, functions that return true/false) and objects (things that fall under concepts).

"A concept is a function whose value is always a truth-value."
  • Object: The Moon, Julius Caesar, the number 4 — things we can refer to.
  • Concept: "is a planet", "is even", "is greater than 3" — predicates that objects satisfy or don't.

Idea 3: Numbers Attach to Concepts, Not Objects

This is one of Frege's most important insights. When we say "there are 3 apples on the table", the number 3 is not a property of the apples — it is a property of the concept "apple on the table".

"The content of a statement of number is an assertion about a concept."
— Grundlagen, §46

The same physical pile can be "1 deck of cards" or "52 cards" — the number depends on which concept you apply.

Idea 4: Extensions of Concepts

The extension of a concept is the collection of all objects that fall under it. Two concepts have the same extension if and only if exactly the same objects fall under both.

The extension of "is a planet in our solar system" = {Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune}.

Frege uses extensions to define numbers: the number belonging to a concept F is the extension of the concept "equinumerous with F".

Idea 5: Hume's Principle (Equinumerosity)

Two concepts have the same number if and only if there exists a one-to-one correspondence (bijection) between their extensions.

"The number which belongs to the concept F is the same as the number which belongs to the concept G if and only if F and G are equinumerous."
— Grundlagen, §63

This is now called Hume's Principle (HP). Frege uses it as the key criterion for when two concepts share a number — without needing to say what numbers are first.

Idea 6: Defining 0, 1, 2, 3…

With all the machinery in place, Frege defines the natural numbers purely logically:

  • 0 = the number belonging to the concept "is not identical with itself" (nothing satisfies this — the extension is empty).
  • 1 = the number belonging to the concept "is identical with 0" (exactly one thing: 0 itself).
  • 2 = the number belonging to the concept "is identical with 0 or identical with 1".
  • And so on: each new number is defined using the numbers already defined.
"0 is the number which belongs to the concept 'not identical with itself'."
— Grundlagen, §74

Idea 7: The Context Principle

One of Frege's three foundational principles stated in the introduction:

"Never ask for the meaning of a word in isolation, but only in the context of a sentence."
— Grundlagen, Introduction

We don't need to point to some mysterious object "3" to give it meaning. We only need to fix the meaning of every sentence in which "3" appears. If we can determine the truth of "3 = 3", "3 > 2", "3 is prime", etc., then "3" has a perfectly determinate meaning.
