Was ist Homotopietypentheorie (HoTT)?

Die Homotopietypentheorie verbindet Typentheorie (eine Grundlage der Mathematik und Informatik) mit Homotopietheorie (einem Teilgebiet der Topologie). Das Ergebnis ist ein mächtiger Formalismus, in dem:

  • Typen als Räume interpretiert werden
  • Terme als Punkte in diesen Räumen
  • Gleichheitsbeweise als Pfade zwischen Punkten
  • Beweise von Gleichheit zwischen Gleichheiten als Homotopien zwischen Pfaden

Warum HoTT?

Konstruktive Mathematik

Jeder Beweis ist gleichzeitig ein Algorithmus. Wenn du beweist, dass etwas existiert, hast du es auch konstruiert.

Computerverifizierbar

Beweise können von Beweisassistenten wie Agda oder Coq maschinell überprüft werden.

Univalenz-Axiom

Isomorphe Strukturen sind gleich. Das formalisiert eine Praxis, die Mathematiker schon immer verwenden.

Höhere Induktive Typen

Pfade = Gleichheitsbeweise

In HoTT ist der Gleichheitstyp IdA(a, b) der zentrale Begriff. Ein Term p : IdA(a, b) ist ein Beweis, dass a und b gleich sind - und gleichzeitig ein Pfad von a nach b im Raum A.

-- Der Gleichheitstyp (Identity Type / Path Type)
data Id {A : Type} : A → A → Type where
  refl : (a : A) → Id a a -- Reflexivität: konstanter Pfad

Pfad-Operationen

Verkettung (Transitivität)

Wenn p : a = b und q : b = c, dann p · q : a = c

Inverse (Symmetrie)

Wenn p : a = b, dann p⁻¹ : b = a

Pfad-Algebra: Die Gruppoide-Struktur

Pfade bilden ein ∞-Gruppoid. Probiere die Gesetze aus:

refl · p = p

Transport entlang eines Pfades

Wenn wir einen abhängigen Typ B : A → U und einen Pfad p : a =A b haben, können wir Elemente von B(a) nach B(b) transportieren:

transport : Π(B : A → U). Π(p : a =A b). B(a) → B(b)

Homotopien: Pfade zwischen Pfaden

Da Gleichheitsbeweise selbst Terme sind, können wir nach Gleichheit zwischen Gleichheitsbeweisen fragen. Ein Beweis α : p =a=b q ist eine Homotopie zwischen den Pfaden p und q.

-- Pfade zwischen Pfaden (2-Pfade)
α : p =a =A b q

-- Das ist eine stetige Deformation von p zu q!

Das Univalenz-Axiom

Das Herzstück von HoTT: Äquivalente Typen sind gleich.

-- Für alle Typen A, B im Universum U:
ua : (A ≃ B) → (A =U B)

-- Und diese Abbildung ist selbst eine Äquivalenz!
univalence : isEquiv(ua)
