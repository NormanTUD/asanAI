1. What Is Persistent Homology?

The big idea in 60 seconds

📍 Start with Points

You have a cloud of data points. Maybe sensor readings, pixel coordinates, or high-dimensional features projected down. The question: what is the "shape" of this data?

🫧 Grow Balls

Imagine inflating a ball around each point. As the radius ε grows, balls start overlapping. When two overlap, connect those points with an edge. When three mutually overlap, fill in a triangle. This builds a simplicial complex.

🕳️ Track Features

As ε increases, topological features appear and disappear. Connected components merge. Loops form then get filled in. We track the birth and death of each feature.

📊 Barcode / Diagram

Each feature becomes a bar: born at some ε, dying at another. Long bars = real structure. Short bars = noise. This is the barcode. Equivalently, plot (birth, death) to get a persistence diagram.

The three types of features we track:
H₀ (Connected Components) — clusters of points. Start with n points = n components. They merge as ε grows.
H₁ (Loops / Holes) — 1-dimensional holes, like the hole in a donut. A ring of edges with no filled interior.
H₂ (Voids) — 2-dimensional cavities, like the inside of a hollow sphere. (We'll focus on H₀ and H₁ in 2D.)

3. Persistence Diagram

The same information as the barcode, plotted differently. Each feature is a point at (birth, death). Points far from the diagonal are "persistent" = significant.

Reading the Diagram:

• Every point represents a topological feature
• x-axis = birth time (ε when it appeared)
• y-axis = death time (ε when it disappeared)
• All points lie above the diagonal (things die after they're born)
• Points far from the diagonal = long-lived = significant
• Points near the diagonal = short-lived = noise
• Green = H₀, Red = H₁

5. Why Does This Matter?

Persistent homology gives you a multi-scale summary of shape that is stable and coordinate-free.

🧬 Biology

Detect loops in protein structures, classify cell shapes, analyze neural spike train data topology.

📡 Sensor Networks

Find coverage holes in sensor networks without knowing exact sensor positions — just connectivity.

🖼️ Image Analysis

Sublevel set persistence on grayscale images detects blobs, ridges, and texture patterns robustly.

📈 Machine Learning

Use persistence diagrams as features for classification. Topological features are invariant to rotation, translation, and small perturbation.

Stability Theorem (Cohen-Steiner, Edelsbrunner, Harer 2007):
Small changes in the input data produce small changes in the persistence diagram (measured by bottleneck distance). This is why persistent homology is robust to noise — it's stable.
