§1 Persistent Homology of Activation Clouds

TDA on hidden-state point clouds — what topological features persist across filtration scales?

Hidden states from transformer layers form point clouds in ℝd. Persistent homology tracks which topological features (connected components β₀, loops β₁, voids β₂) appear and die as we grow ε-balls around each point. Features that persist across many scales encode genuine structure, not noise.

§2 Mapper Algorithm / Reeb Graphs

Compressing high-dimensional activation manifolds into interpretable graphs

The Mapper algorithm: pick a filter function f (e.g., first principal component of activations), cover its range with overlapping intervals, cluster within each preimage, then connect clusters that share points. The result is a Reeb graph — a skeleton of the underlying manifold.

This is more informative than UMAP because it preserves topological connectivity, not just local distances.

§3 Manifold Hypothesis & Intrinsic Dimensionality

Hidden states live on low-dimensional manifolds embedded in high-dimensional space

A 768-dim residual stream doesn't use all 768 dimensions equally. The intrinsic dimension dint measures the "true" dimensionality of the data manifold. Methods: local PCA (count significant singular values), correlation dimension, MLE estimators.

LLM connection: Intrinsic dimensionality of hidden states varies by layer and by input type. Early layers: high dint (raw token features). Middle layers: dint often drops (compression/abstraction). Late layers: dint rises again (decompression for prediction). Measuring this per-layer gives a "dimensionality profile" that characterizes what each layer does. Superposition means dint > dambient in some sense — features are packed into fewer dimensions than they "should" need.

§4 Fiber Bundles & the Residual Stream

The residual stream as a vector bundle over a base space of "positions"

Think of a fiber bundle: base space B (token positions / abstract "task" coordinates), fiber F (the residual stream vector at each position), and a projection π: E → B. An attention head defines a connection — it parallel-transports information between fibers. A non-trivial bundle means you can't globally decompose the representation into independent position-dependent and content-dependent parts.

LLM connection: The residual stream at position i is a fiber over the "position" base point. Attention = connection that transports information between fibers. Holonomy (the rotation after parallel transport around a loop) measures how much the representation is path-dependent — i.e., the order of attention operations matters. Non-zero holonomy ↔ non-commuting attention heads. This framework from Geometry of Truth (Marks et al.) helps explain why representation surgery must respect the bundle structure.

§5 Morse Theory on Activation Landscapes

Critical points of scalar functions on the hidden-state manifold

Given a function f on the activation manifold (e.g., loss, probe accuracy, a single neuron's activation), Morse theory classifies critical points by their index (number of negative eigenvalues of the Hessian). As you sweep a level set f = c upward, the topology changes only at critical points: index-0 = new component born, index-1 = handle attached (loop created or components merged), index-2 = cavity filled.

Click to place critical points. Drag the level-set slider to see how topology changes. The Euler characteristic χ = #(minima) − #(saddles) + #(maxima) is a topological invariant.

LLM connection: Think of a linear probe's accuracy as a Morse function on the manifold of hidden states. Critical points reveal phase transitions in what information is linearly accessible. A saddle point in probe-accuracy landscape means there's a direction where the representation is "between" two interpretable states — this is where superposition lives. Morse theory also applies to the loss landscape: the number and index of critical points constrains how many qualitatively different solutions (modes) exist.

§6 Sheaves on Attention Graphs

Sheaf cohomology detects global inconsistencies in local attention patterns

A cellular sheaf on a graph assigns a vector space to each node and edge, with restriction maps between them. On an attention graph: nodes = tokens (with their hidden states), edges = attention connections (with projected key-query subspaces). The sheaf Laplacian generalizes the graph Laplacian. Its kernel = global sections = globally consistent information. Non-trivial cohomology H¹ = obstructions to global consistency.

Click nodes to change their local vector. Click edges to change restriction maps. Watch the coboundary and cohomology update.

LLM connection: Sheaf theory on the attention graph captures whether attention heads produce globally coherent representations. dim(H⁰) = dimension of information all tokens agree on (global context). dim(H¹) = number of independent "inconsistency modes" — information that's locally present but can't be made globally consistent. This is precisely what happens in hallucination: local attention patterns are fine but globally they're contradictory. The sheaf Laplacian spectrum also predicts information diffusion speed through the attention graph.

§7 Topology of Superposition & Feature Geometry

How more features than dimensions pack together — polytope geometry and interference

Superposition: N features encoded in d < N dimensions. Each feature is a unit vector in ℝd. The interference between features i,j is |⟨fᵢ,fⱼ⟩|. Optimal packings minimize maximum interference and correspond to known geometric configurations: orthogonal bases, simplices, cross-polytopes, etc.

LLM connection: The Anthropic superposition paper shows that feature vectors arrange into specific polytope geometries depending on sparsity. At high sparsity, you can pack exponentially many features (simplex/cross-polytope). The topological insight: the space of optimal packings itself has interesting topology — there are discrete symmetry groups (the polytope's symmetry group) and continuous families of equivalent solutions. Phase transitions between packing geometries as sparsity changes correspond to topological changes in the solution manifold.

§8 Topological Complexity & Configuration Spaces

TC(X) measures how hard it is to plan continuous paths — relevant for sequential processing

Topological complexity TC(X) is the minimum number of "motion planning rules" needed to continuously navigate a space X. For the circle S¹: TC = 2 (you need one rule for most pairs, another for antipodal points). For Sn: TC = 2 if n odd, 3 if n even. Higher TC = harder to continuously interpolate.

LLM connection: TC measures how many "strategies" a transformer needs to continuously route information. The configuration space of n tokens (no two at the same position in representation space) is related to the braid group Bn. Attention patterns that swap token information correspond to braid generators σᵢ. The topological complexity of this configuration space lower-bounds the number of distinct attention patterns needed. This connects to the circuit complexity of transformers and why certain tasks require multiple layers.

§9 Circular Coordinates from Persistent Cohomology

Detecting and parameterizing circular features in hidden states

When persistent H¹ detects a loop, persistent cohomology gives you a circular coordinate θ: X → S¹ that parameterizes it. This is like finding the "phase" of a circular feature (time of day, compass direction, sentiment cycle) directly from the point cloud.

Arrange points in a rough circle. The algorithm finds the cohomology class and computes θ for each point. Color shows the recovered circular coordinate. Compare with ground truth.

LLM connection: Circular coordinates extracted from hidden states directly reveal periodic features the model has learned: days of the week, months, hours, compass directions, modular arithmetic. This was used in the "Geometry of Truth" line of work. The key insight is that you don't need to know what the circular feature is — cohomology finds it automatically from the topology of the activation cloud. If H¹ is trivial, there's no circular structure (the model uses a linear, not circular, representation).
