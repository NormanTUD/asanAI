WQ — Query Projection

Transforms each embedding into a "question vector." Words with similar Q vectors are looking for the same things.

WK — Key Projection

Transforms each embedding into a "label vector." When Qi · Kj is large, word i attends to word j — they form an edge.

WV — Value Projection

The actual content that gets sent along edges. After Q·K determines the structure (which simplices exist), V determines what meaning flows through that structure.

How it works:
1. Compute Qi · Kj for all pairs → attention score
2. Softmax → attention weight αij
3. If min(αij, αji) > ε → edge {i,j} exists
4. If all 3 edges of a triangle exist → 2-simplex {i,j,k} exists
5. V vectors flow along these connections → output meaning
