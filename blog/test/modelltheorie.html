🧠 Modelltheorie – Interaktiv erklärt

Was ist Modelltheorie?

Stell dir vor, du hast Regeln (wie Spielregeln) und fragst dich: „Gibt es eine Welt, in der alle diese Regeln gleichzeitig gelten?"

Die Modelltheorie untersucht genau das: Sie verbindet formale Aussagen (Syntax) mit konkreten Strukturen (Semantik), in denen diese Aussagen wahr sind.

Eine solche Struktur nennt man ein Modell.

4. Die Signatur – Der „Bauplan"

📋

Analogie: Die Signatur ist wie die Spielanleitung, die sagt: „In diesem Spiel gibt es Figuren, eine Würfel-Funktion und eine Freundschafts-Beziehung." Sie legt fest, welche Arten von Symbolen es gibt – aber noch nicht, was sie konkret bedeuten.

σ = (R, F, K, Ar)

R = Relationssymbole (z.B. „<", „Freund")
F = Funktionssymbole (z.B. „+", „Nachfolger")
K = Konstantensymbole (z.B. „0", „Alice")
Ar = Stelligkeit (wie viele Argumente jedes Symbol braucht)

Beispiel: Signatur für Graphen: σ = ({E}, {}, {}, {E↦2})

→ Ein Relationssymbol E mit Stelligkeit 2 (Kanten verbinden je 2 Knoten)

5. Vollständiges Beispiel: Dichte Ordnung

📏

Reales Beispiel: Die rationalen Zahlen (ℚ) mit der Kleiner-Relation (<) sind ein Modell für die Theorie der „dichten Ordnungen".

Axiom 1: Trichotomie

∀x,y: (x < y) ∨ (x = y) ∨ (y < x)

„Für je zwei Zahlen gilt: entweder ist die eine kleiner, oder sie sind gleich, oder die andere ist kleiner."

✓ Gilt für ℚ: z.B. 1/3 < 1/2, oder 0.5 = 1/2, oder 3 > 2

Axiom 2: Antisymmetrie

∀x,y: x < y → ¬(y < x)

„Wenn x kleiner als y ist, dann ist y NICHT kleiner als x."

✓ Gilt für ℚ: Wenn 2 < 5, dann ist 5 nicht < 2.

Axiom 3: Transitivität

∀x,y,z: (x < y ∧ y < z) → x < z

„Wenn x < y und y < z, dann auch x < z."

✓ Gilt für ℚ: 1 < 3 und 3 < 7, also 1 < 7. ✓

Axiom 4: Offenheit (kein Rand)

∀x: ∃y,z: (y < x ∧ x < z)

„Zu jeder Zahl gibt es eine kleinere UND eine größere."

✓ Gilt für ℚ: Zu 5 gibt es 4 (kleiner) und 6 (größer).

✗ Gilt NICHT für {1,2,3}: 1 hat nichts Kleineres!

Axiom 5: Dichtheit

∀x,y: x < y → ∃z: (x < z ∧ z < y)

„Zwischen je zwei verschiedenen Zahlen liegt immer noch eine weitere."

✓ Gilt für ℚ: Zwischen 1/3 und 1/2 liegt z.B. 5/12.

✗ Gilt NICHT für ℤ (ganze Zahlen): Zwischen 2 und 3 liegt keine ganze Zahl!

📝 Zusammenfassung

Universum Die Menge der „Dinge", über die wir reden
Signatur Der Bauplan: welche Relationen, Funktionen, Konstanten es gibt
Struktur Universum + konkrete Relationen/Funktionen/Konstanten
Interpretation Zuordnung: Symbole der Sprache → Elemente der Struktur
Modell Eine Struktur, die alle Aussagen einer Theorie erfüllt
𝒜 ⊨ T „Struktur 𝒜 ist ein Modell von Theorie T"
