② Step-by-Step Forward Pass

Pick an input x and watch every number flow through the network.

1Dense₁: z = W₁·x + b₁ — each neuron computes an affine function

2ReLU: h = max(0, z) — bend each affine piece at zero

3Dense₂: ŷ = W₂·h + β — glue all pieces into one output
