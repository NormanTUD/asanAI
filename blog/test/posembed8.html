Positional Embedding Helices — 3 Wörter im 3D-Raum

Jedes Wort hat ein festes Embedding (◆). Die Spirale zeigt, wie sich das Wort durch sinusoidales Positional Encoding an jeder Sequenzposition verschiebt.

PE(pos, 2i) = sin(pos / 10000^(2i/d_model)) PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model)) Embedding an Position p = e(word) + PE(p)

◆ = Rohes Wort-Embedding (ohne PE)  |  Spirale = Wort + PE(pos) für jede Position  |  Farbe = Positionsnummer
Separation-Slider steuert den Abstand der Basis-Embeddings zueinander.
