§1 Jacobian Decomposition Qed ✓

Any matrix J decomposes into a symmetric part J_sym = (J + Jᵀ)/2 and an antisymmetric part J_antisym = (J − Jᵀ)/2. The Coq proof shows this holds for all indices i,j < n by simple field arithmetic.

Theorem jacobian_decomposition_general : forall (n : nat) (J : Mat), mat_eq n J (mat_add (mat_sym J) (mat_antisym J)). Proof. intros n J i j Hi Hj. unfold mat_add, mat_sym, mat_antisym. field. Qed.

§2 Symmetry & Antisymmetry Properties Qed ✓

J_symᵀ = J_sym (symmetric part is symmetric) and J_antisymᵀ = −J_antisym (antisymmetric part is antisymmetric). Visually: the symmetric heatmap is mirror-symmetric about the diagonal; the antisymmetric one is sign-flipped.

Theorem sym_is_symmetric_general : forall (n : nat) (J : Mat), mat_eq n (mat_transpose (mat_sym J)) (mat_sym J). Proof. intros n J i j Hi Hj. unfold mat_transpose, mat_sym. field. Qed. Theorem antisym_is_antisymmetric_general : forall (n : nat) (J : Mat), mat_eq n (mat_transpose (mat_antisym J)) (mat_neg (mat_antisym J)). Proof. intros n J i j Hi Hj. unfold mat_transpose, mat_antisym, mat_neg. field. Qed.

§3 Trace Properties Qed ✓

The trace is additive, homogeneous, and the divergence (tr(J)) comes entirely from the symmetric part. The antisymmetric part always has zero trace. Drag the slider to scale J and watch the traces.

Theorem divergence_from_symmetric_part_general : forall (n : nat) (J : Mat), trace_n n J = trace_n n (mat_sym J). Theorem antisym_trace_zero_general : forall (n : nat) (J : Mat), trace_n n (mat_antisym J) = 0. Theorem trace_scale_general : forall (n : nat) (c : R) (A : Mat), trace_n n (mat_scale c A) = c * trace_n n A.

§4 Frobenius Orthogonality Qed ✓

The symmetric and antisymmetric parts are orthogonal under the Frobenius inner product: ⟨J_sym, J_antisym⟩_F = tr(J_symᵀ · J_antisym) = 0. This means the decomposition is a genuine orthogonal projection. The proof uses a swap-and-cancel argument on the double sum.

Theorem frobenius_sym_antisym_orthogonal : forall (n : nat) (J : Mat), frobenius_inner n (mat_sym J) (mat_antisym J) = 0. Proof. (* Key insight: each (i,k) term cancels with (k,i) term *) (* Σ_i Σ_k (J(k,i)² - J(i,k)²)/4 is antisymmetric under i↔k swap *) (* So the sum + its swap = 0, hence the sum itself = 0 *) Theorem frobenius_sym_antisym_orthogonal : forall (n : nat) (J : Mat), frobenius_inner n (mat_sym J) (mat_antisym J) = 0. Proof. intros n J. unfold frobenius_inner. unfold trace_n, mat_mult, mat_transpose, mat_sym, mat_antisym. rewrite (sum_f_ext _ (fun i => sum_f (fun k => (J k i * J k i - J i k * J i k) / 4) n) n). 2: { intros i Hi. apply sum_f_ext. intros k Hk. field. } assert (Hkey : sum_f (fun i => sum_f (fun k => (J k i * J k i - J i k * J i k) / 4) n) n + sum_f (fun i => sum_f (fun k => (J k i * J k i - J i k * J i k) / 4) n) n = 0). { rewrite (sum_f_swap (fun i k => (J k i * J k i - J i k * J i k) / 4) n n) at 1. rewrite <- sum_f_add. rewrite (sum_f_ext _ (fun _ => 0) n). 2: { intros k Hk. rewrite <- sum_f_add. rewrite (sum_f_ext _ (fun _ => 0) n). 2: { intros i Hi. lra. } apply sum_f_zero. } apply sum_f_zero. } lra. Qed. Qed.

§5 Idempotence & Cross-Annihilation Qed ✓

Applying sym twice gives sym (it's a projector). Applying sym to an antisymmetric matrix gives zero, and vice versa. This confirms the decomposition is a pair of complementary orthogonal projections.

§6 Shear Tensor is Trace-Free Qed ✓

The shear tensor S = J_sym − (tr(J_sym)/n)·I captures pure shape-change with no volume change. The proof shows tr(S) = tr(J_sym) − tr(J_sym) = 0. Adjust the matrix and see the shear.

Definition mat_shear (n : nat) (A : Mat) : Mat := fun i j => mat_sym A i j - (trace_n n (mat_sym A) / INR n) * (if Nat.eq_dec i j then 1 else 0). Theorem shear_trace_free_general : forall (n : nat) (J : Mat), (n > 0)%nat -> trace_n n (mat_shear n J) = 0.

§7 Residual Connection: J = I + Δ Qed ✓

A residual layer maps h ↦ h + Δ(h), so its Jacobian is I + J_Δ. The divergence is n + tr(J_Δ), and the rotation (antisymmetric part) comes entirely from Δ — the identity contributes nothing.

Theorem residual_divergence_general : forall (n : nat) (J_delta : Mat), trace_n n (mat_add mat_id J_delta) = INR n + trace_n n J_delta. Theorem residual_antisym_general : forall (n : nat) (J_delta : Mat), mat_eq n (mat_antisym (mat_add mat_id J_delta)) (mat_antisym J_delta).

§8 Trace Cyclicity Qed ✓

tr(AB) = tr(BA), and for three matrices: tr(ABC) = tr(BCA) = tr(CAB). The proof uses a double-sum swap lemma and matrix multiplication associativity. Randomize matrices and verify the identity holds numerically.

Theorem trace_cyclic_3_general : forall (n : nat) (A B C : Mat), trace_n n (mat_mult n A (mat_mult n B C)) = trace_n n (mat_mult n C (mat_mult n A B)). Proof. (* Uses: tr(XY) = tr(YX) + associativity *) Theorem trace_cyclic_3_general : forall (n : nat) (A B C : Mat), trace_n n (mat_mult n A (mat_mult n B C)) = trace_n n (mat_mult n C (mat_mult n A B)). Proof. intros n A B C. (* Step 1: tr(A(BC)) = tr((BC)A) by tr(XY) = tr(YX) *) rewrite (trace_mult_commutative_general n A (mat_mult n B C)). (* Now goal: tr((BC)A) = tr(C(AB)) *) (* Step 2: Show (BC)A =_eq A(BC) ... no, we need associativity *) (* tr((BC)A) -- use assoc to get tr(B(CA)) *) unfold trace_n. rewrite (sum_f_ext _ (fun i => mat_mult n B (mat_mult n C A) i i) n). 2: { intros i Hi. apply (mat_mult_assoc_general n B C A i i Hi Hi). } (* Now goal: sum_f (fun i => mat_mult n B (mat_mult n C A) i i) n = sum_f (fun i => mat_mult n C (mat_mult n A B) i i) n *) fold (trace_n n (mat_mult n B (mat_mult n C A))). fold (trace_n n (mat_mult n C (mat_mult n A B))). (* tr(B(CA)) = tr((CA)B) = tr(C(AB)) *) rewrite (trace_mult_commutative_general n B (mat_mult n C A)). unfold trace_n. rewrite (sum_f_ext _ (fun i => mat_mult n C (mat_mult n A B) i i) n). 2: { intros i Hi. apply (mat_mult_assoc_general n C A B i i Hi Hi). } reflexivity. Qed. (* tr(ABC) = tr(BCA) *) Theorem trace_cyclic_3_general' : forall (n : nat) (A B C : Mat), trace_n n (mat_mult n A (mat_mult n B C)) = trace_n n (mat_mult n B (mat_mult n C A)). Proof. intros n A B C. rewrite trace_cyclic_3_general. rewrite trace_cyclic_3_general. reflexivity. Qed. Qed.

§9 Orthogonal Similarity Invariance Qed ✓

If Q is orthogonal (QQᵀ = I), then tr(QᵀJQ) = tr(J). Divergence is invariant under rotation of coordinates. Adjust the rotation angle θ for a 2D rotation matrix Q and see that the trace of QᵀJQ never changes.

Theorem trace_orthogonal_similarity : forall (n : nat) (Q J : Mat), mat_eq n (mat_mult n Q (mat_transpose Q)) mat_id -> trace_n n (mat_mult n (mat_transpose Q) (mat_mult n J Q)) = trace_n n J.
