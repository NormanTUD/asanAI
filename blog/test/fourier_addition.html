🧠 How a Neural Network Learns Modular Addition

The "Fourier Multiplication Algorithm" — explained step by step

Step 0: What Problem Are We Solving?

Imagine you have a clock with P hours on it (instead of 12). We want to add two numbers, but the answer "wraps around" when it gets too big.

Example with P = 7:
3 + 5 = 8, but 8 wraps around to 1 (because 8 - 7 = 1)
2 + 4 = 6, stays as 6 (because 6 < 7)
6 + 6 = 12, wraps to 5 (because 12 - 7 = 5)

A neural network learned to do this — and the way it figured it out is beautiful. It uses waves and circles!

Step 1: Numbers Live on a Circle (Clock)

Since our numbers wrap around, they naturally live on a circle — just like hours on a clock!

Each number gets a position on the circle. Adding two numbers means moving around the circle by that many steps.

Step 2: What Is a Wave?

A wave is a pattern that repeats. Think of it like a swing going back and forth, or ripples in water.

A wave has:
• Height — how tall it goes (between -1 and 1)
• Frequency — how many times it repeats (how "fast" it wiggles)
• Position — where you are along the wave

The neural network uses waves to encode numbers. Move the sliders to see how waves change!

Step 3: The Network Puts Numbers on Circles

Here's the clever trick: the network takes each number and places it as a point on a circle.

For a number a and a frequency k:
• The horizontal position is the wave height at position a
• The vertical position is a shifted wave height at position a

Different frequencies spread the numbers differently around the circle!

The network uses several frequencies at once (like k=14, 35, 41, 42, 52 for P=113). Each gives a different "view" of the numbers.

Step 4: Adding = Combining Circle Positions

Here's the magic! When you put number a on the circle and number b on the circle, their combined rotation gives you a + b mod P!

The network computes this by multiplying the wave values together:
• (wave of a) × (wave of b) − (shifted wave of a) × (shifted wave of b)
= wave of (a+b)

This is how the network "adds" without actually adding — it rotates!

Watch: the red angle (a) plus the green angle (b) equals the gold angle (a+b mod P).

Step 5: Reading Off the Answer (Constructive Interference)

Now the network has the combined rotation for (a+b). But how does it pick the right answer?

For each possible answer c (0, 1, 2, ... P-1), the network computes a score.

The score is highest when c = a+b mod P, because the waves line up (constructive interference).
For wrong answers, the waves cancel out (destructive interference).

Using multiple frequencies makes the correct answer stand out even more!

The bar chart shows the score for each possible answer. The tallest bar is the network's answer!

Step 6: The Full Algorithm in Action

Let's see all the steps together! The network:

1. Takes numbers a and b
2. Places them on circles at several frequencies
3. Combines the rotations to get (a+b) on each circle
4. For each possible answer c, checks how well the waves line up
5. Picks the c with the highest total score → that's a+b mod P!
