Reminder: A fibre bundle E →π B assigns to each point x ∈ B a fibre F. A section σ: B → E picks one point in each fibre.

Key question: What if we care not just about the value of a section at a point, but also about its derivative (and higher derivatives)? That's exactly what jet bundles capture!

The Core Idea of Jet Bundles

Imagine multiple sections passing through the same point. Some just touch (same value), some are tangent (same value AND derivative), and some agree even more closely (same higher derivatives).

Base point x₀
Jet = Taylor data. The k-jet of a section σ at x is the equivalence class of all sections that agree with σ up to order k at x.

• 0-jet: same value → σ₁(x) = σ₂(x)
• 1-jet: same value + same first derivative → σ₁(x) = σ₂(x) and σ₁'(x) = σ₂'(x)
• k-jet: agreement up to all derivatives of order ≤ k

The jet bundle JkE is the bundle whose fibre over x consists of all possible k-jets at x.

The First Jet Bundle J¹E

For a bundle E → ℝ with fibre ℝ, a 1-jet at x is a pair (σ(x), σ'(x)) — a value and a slope. Drag the point and the tangent line to explore the fibre of J¹E.

What you're seeing:
• The blue dot on the base is x.
• The pink dot above it is the value σ(x) — this lives in the original fibre.
• The green tangent line represents the slope σ'(x).
• Together, (x, σ(x), σ'(x)) is one point in J¹E.

Fibre dimension: Original fibre is 1D (the value). The 1-jet fibre is 2D (value + slope). In general, dim(J¹E) = dim(B) + dim(F) + dim(B)·dim(F).

Higher Jet Bundles JkE

As we go to higher orders, each jet carries more Taylor data. Watch how curves that share more derivatives stay closer together near the base point.

The pattern:
• J⁰E: fibre records (value) — 1 number
• J¹E: fibre records (value, 1st derivative) — 2 numbers
• J²E: fibre records (value, 1st deriv, 2nd deriv) — 3 numbers
• JkE: fibre records k+1 numbers (for 1D base, 1D fibre)

In general for dim B = n, dim F = m: dim JkF = n + m · C(n+k, k)

There's a natural tower: … → J²E → J¹E → J⁰E ≅ E → B where each projection forgets the highest derivative.

Equivalence Classes — What "Same Jet" Means

Two sections have the same k-jet at x₀ if they're indistinguishable up to order k. Click to place sections and see which ones are equivalent.

Formally: Two local sections σ₁, σ₂ of E → B have the same k-jet at x₀ iff in any (hence every) local coordinate system:
∂|α|σ₁/∂xα(x₀) = ∂|α|σ₂/∂xα(x₀) for all |α| ≤ k This is coordinate-independent! The equivalence class jkx₀σ is the k-jet of σ at x₀.

Why it matters: PDEs can be formulated as submanifolds of jet bundles. A PDE of order k is a subset R ⊂ JkE, and solutions are sections whose jet prolongation lands in R.
