⚙ Die Unendlichkeitsmaschine

Jedes Zahnrad dreht sich exponentiell langsamer als das vorherige — und sie greifen ineinander. Das schnellste unterscheidet benachbarte Positionen. Das langsamste braucht tausende für eine Umdrehung. Zusammen erzeugen sie ein einzigartiges Interferenzmuster — den Fingerabdruck jeder Position.

⚙ Wie eine echte Unendlichkeitsmaschine: Die Zahnräder greifen ineinander — jedes treibt das nächste an, aber dreht sich langsamer. Benachbarte Zahnräder drehen sich in entgegengesetzte Richtungen, genau wie bei echten Getrieben.

Positional Encoding: Jedes Zahnrad ist eine Frequenzdimension. Die Formel: PE(pos, 2i) = sin(pos / 10000^(2i/d)), PE(pos, 2i+1) = cos(pos / 10000^(2i/d)). Die Frequenzen bilden eine geometrische Progression — genau wie die Übersetzungsverhältnisse.

Das Interferenzfeld: Jeder Quellpunkt sendet Wellen aus. Wo sich alle konstruktiv überlagern → hell. Dieses Muster ist für jede Position einzigartig.

Warum sin & cos? cos(a−b) = cos(a)cos(b) + sin(a)sin(b) → die Ähnlichkeit zweier Positionen hängt nur vom Abstand ab, nicht von der absoluten Position.
