📐 How It Works

  • Each token position is hashed through a chaotic spatial map (Arnold's cat map variant)
  • The hash determines a focus point on the image
  • Regions near focus points are sharp; distant regions are pixelated
  • Total displacement of all tokens from defaults = sway intensity
  • Each layer adds more warping complexity
  • The Jacobi matrix shows local space distortion per layer
