📐 Layer-Progressive Warping

  • Layer 0: nearly flat grid — only DC / lowest frequencies
  • Layer 1: slight bending — low-freq FFT bands added
  • Layer 2–4: progressively more warped — higher freq bands accumulate
  • The Jacobi matrix J(layer) grows with layer depth
  • All stalks at the same layer share the same warped space
  • Each stalk contributes its own FFT band within that shared space

warp(x, layer) = growth^layer × J(x) × ∇FFT_cumulative(layer)
