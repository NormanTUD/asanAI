What is Catastrophe Theory?

Catastrophe theory, developed by French mathematician René Thom in the 1960s–70s, studies how small, smooth changes in control parameters can cause sudden, discontinuous jumps in the state of a system. It's a branch of bifurcation theory within dynamical systems and singularity theory in mathematics.

Core Idea

Consider a system described by a potential function V(x; a, b, …) where x is the state variable and a, b, … are control parameters. The system naturally sits at minima of V (stable equilibria). As you smoothly change the control parameters, the shape of V changes — and at critical moments, a minimum can disappear, forcing the system to jump to a different minimum. That sudden jump is the catastrophe.

Thom's Classification Theorem

Thom proved that for systems with ≤ 2 state variables and ≤ 4 control parameters, there are exactly 7 elementary catastrophes (up to smooth equivalence). The four "cuspoid" catastrophes (1 state variable) are explored here:

Key Concepts

  • Equilibria: Points where dV/dx = 0 (critical points of the potential)
  • Stability: Minima of V are stable; maxima are unstable
  • Bifurcation Set: The set of control parameter values where the number/type of equilibria changes
  • Hysteresis: The system's state depends on its history — path-dependent jumps
  • Catastrophe Manifold: The surface dV/dx = 0 in (x, params) space

Fold Catastrophe (A₂)

1 state variable Codimension 1 Simplest catastrophe
V(x) = x³ + a·x

The fold is the simplest catastrophe. For a < 0, V(x) has both a local min and max (two equilibria). At a = 0, they merge into an inflection point. For a > 0, there are no equilibria — the potential is monotonic. The single equilibrium simply vanishes.

Bifurcation Diagram & Equilibria

The catastrophe manifold is the curve dV/dx = 3x² + a = 0, i.e., x = ±√(-a/3). It exists only for a ≤ 0. The blue curve shows stable equilibria (minima), red shows unstable (maxima). At a = 0, they collide and annihilate — the fold point.

Cusp Catastrophe (A₃)

1 state variable Codimension 2 Most famous catastrophe
V(x) = x⁴ + a·x² + b·x

The cusp is the most widely applied catastrophe. It features bimodality (two competing stable states), hysteresis (path-dependent jumping), and divergence (small changes in b cause huge differences when a < 0). The bifurcation set in (a,b)-space forms a cusp-shaped curve.

Swallowtail Catastrophe (A₄)

1 state variable Codimension 3 Named for its bifurcation shape
V(x) = x⁵ + a·x³ + b·x² + c·x

The swallowtail adds a third control parameter. Its bifurcation set in (a, b, c)-space has a characteristic swallowtail shape. For certain parameter values, up to 3 stable states can coexist, enabling richer transition behavior.

Butterfly Catastrophe (A₅)

1 state variable Codimension 4 Highest cuspoid elementary catastrophe
V(x) = x⁶ + a·x⁴ + b·x³ + c·x² + d·x

The butterfly catastrophe has 4 control parameters and can exhibit up to 3 coexisting stable states separated by 2 unstable ones (5 equilibria total). The "butterfly factor" d controls whether the system prefers a compromise middle state or extreme states.
