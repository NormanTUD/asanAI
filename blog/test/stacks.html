Stacks: Sheaves of Categories

A stack is like a sheaf, but instead of assigning sets of data to each region, it assigns entire categories (often groupoids). Descent works for objects up to isomorphism. Click each layer!

🏛 Groupoid of Objects over U
For each open set U, we get a whole category of objects — not just a set!
🔗 Restriction Functors
Moving from a bigger region to a smaller one? There's a functor for that — pulling back objects.
🧩 Descent / Gluing Condition
Local objects that agree on overlaps (up to isomorphism) glue into a global object. This is the magic!
📐 Base: Topological Space / Site
The "ground" — a space (or Grothendieck site) whose open sets form the indexing system.
✦

Algebraic Stacks

An algebraic stack is a stack over the category of schemes (algebraic geometry's building blocks) that has a smooth atlas — a scheme that maps onto it nicely. It's the ultimate framework for handling moduli problems with symmetries.

Algebraic Stack = Stack + Smooth Atlas + Algebraic Structure

Example: The moduli stack of elliptic curves, ℳ₁,₁, classifies all elliptic curves — but remembers their automorphisms. Every elliptic curve has at least a ℤ/2 symmetry, so this can't be an ordinary space. It's an algebraic stack.
