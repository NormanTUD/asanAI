📖 Was ist Kategorientheorie?

Die Kategorientheorie ist ein Teilgebiet der Mathematik, das sich mit abstrakten Strukturen und den Beziehungen zwischen ihnen beschäftigt. Sie wurde in den 1940er Jahren von Samuel Eilenberg und Saunders Mac Lane entwickelt.

Statt einzelne mathematische Objekte zu untersuchen, betrachtet die Kategorientheorie Muster und Beziehungen zwischen Objekten. Sie wird oft als "Mathematik der Mathematik" bezeichnet.

📦 Kategorien

Eine Kategorie C besteht aus:

  • Einer Klasse von Objekten Ob(C)
  • Für je zwei Objekte A, B einer Menge von Morphismen Hom(A, B)
  • Einer Komposition: Wenn f: A → B und g: B → C, dann existiert g ∘ f: A → C
  • Für jedes Objekt A einem Identitätsmorphismus id_A: A → A

🔀 Morphismen & Komposition

Morphismen (auch Pfeile genannt) sind die Beziehungen zwischen Objekten. Die Komposition muss assoziativ sein und Identitäten respektieren.

Axiome: (h ∘ g) ∘ f = h ∘ (g ∘ f) und f ∘ id = f = id ∘ f

🔄 Funktoren

Ein Funktor F: C → D ist eine strukturerhaltende Abbildung zwischen Kategorien. Er bildet:

  • Objekte auf Objekte ab: A ↦ F(A)
  • Morphismen auf Morphismen ab: f ↦ F(f)
  • Erhält Komposition: F(g ∘ f) = F(g) ∘ F(f)
  • Erhält Identitäten: F(id_A) = id_{F(A)}
Wähle einen Funktor-Typ und beobachte, wie Objekte und Morphismen zwischen Kategorien abgebildet werden.

🌊 Natürliche Transformationen

Eine natürliche Transformation η: F ⇒ G zwischen zwei Funktoren F, G: C → D ist eine Familie von Morphismen η_A: F(A) → G(A) für jedes Objekt A in C, sodass für jeden Morphismus f: A → B das folgende Diagramm kommutiert:

G(f) ∘ η_A = η_B ∘ F(f)

Gehe Schritt für Schritt durch, um zu sehen, wie natürliche Transformationen funktionieren.

✖ Produkte & Koprodukte

Das Produkt zweier Objekte A und B ist ein Objekt A × B zusammen mit Projektionen π₁: A×B → A und π₂: A×B → B, das eine universelle Eigenschaft erfüllt.

Das Koprodukt (duale Konstruktion) ist A ⊔ B mit Inklusionen ι₁: A → A⊔B und ι₂: B → A⊔B.

Beobachte die universelle Eigenschaft: Für jedes Objekt X mit passenden Morphismen existiert ein eindeutiger Morphismus zum Produkt/Koprodukt.

🎯 Limiten & Kolimiten

Limiten verallgemeinern Produkte, Equalizer und Pullbacks. Ein Limes eines Diagramms D: J → C ist ein universeller Kegel über dem Diagramm.

Kolimiten sind die duale Konstruktion (Koprodukte, Koequalizer, Pushouts).

Wähle eine Konstruktion und beobachte die universelle Eigenschaft.

⚖ Adjunktionen

Eine Adjunktion besteht aus zwei Funktoren F: C → D (linksadjungiert) und G: D → C (rechtsadjungiert), geschrieben F ⊣ G, mit einer natürlichen Bijektion:

Hom_D(F(A), B) ≅ Hom_C(A, G(B))

Adjunktionen sind eines der wichtigsten Konzepte der Kategorientheorie. Sie verbinden zwei Kategorien auf besonders harmonische Weise.

🏛 Yoneda-Lemma

Das Yoneda-Lemma besagt: Für jeden Funktor F: Cop → Set und jedes Objekt A in C gibt es eine natürliche Bijektion:

Nat(Hom(—, A), F) ≅ F(A)

Insbesondere: Ein Objekt wird vollständig durch seine Beziehungen zu allen anderen Objekten bestimmt (Yoneda-Einbettung).

"Ein Objekt ist vollständig durch die Morphismen bestimmt, die in es hineinführen." – Das ist die Essenz des Yoneda-Lemmas.

🌀 Monaden

Eine Monade auf einer Kategorie C ist ein Tripel (T, η, μ) wobei:

  • T: C → C ein Endofunktor ist
  • η: Id_C ⇒ T die Einheit (unit) ist
  • μ: T² ⇒ T die Multiplikation (join) ist

Mit den Axiomen: μ ∘ Tμ = μ ∘ μT (Assoziativität) und μ ∘ Tη = μ ∘ ηT = id_T (Einheit)

Monaden sind in der Programmierung (z.B. Haskell) allgegenwärtig. Hier siehst du die kategorientheoretische Grundlage.
