⚙ Die Unendlichkeitsmaschine

Positional Encoding als Zahnradkaskade: Jedes Zahnrad dreht sich exponentiell langsamer als das vorherige. Das schnellste unterscheidet benachbarte Positionen. Das langsamste braucht tausende Positionen für eine Umdrehung. Zusammen erzeugen sie für jede Position ein einzigartiges Interferenzmuster — den Fingerabdruck der Position.

⚙ Die Unendlichkeitsmaschine im Deutschen Museum hat Zahnräder, die sich von Stufe zu Stufe zehnmal langsamer drehen. Das letzte Zahnrad dreht sich einmal in 100 Millionen Jahren.

Genau so funktioniert Positional Encoding: Die Frequenzdimensionen bilden eine geometrische Progression von schnell nach langsam. Dimension 0 (das schnellste Zahnrad) oszilliert bei jeder Position — sie unterscheidet Position 5 von Position 6. Die letzte Dimension (das langsamste Zahnrad) braucht ~10.000 Positionen für eine volle Umdrehung — sie unterscheidet den Anfang eines Textes von der Mitte.

Das Interferenzfeld: Jedes Zahnrad erzeugt an seiner aktuellen Winkelposition eine Welle. Wo sich alle Wellen konstruktiv überlagern → hell. Dieses Muster ist für jede Position einzigartig. Benachbarte Positionen erzeugen ähnliche Muster (= ähnliche Embeddings), weit entfernte völlig verschiedene.

Warum sin & cos? Jedes Zahnrad-Paar nutzt Sinus und Kosinus — das beschreibt einen Punkt auf einem Kreis. Der mathematische Trick: cos(a−b) = cos(a)cos(b) + sin(a)sin(b). Dadurch hängt die Ähnlichkeit zweier Positionen nur vom Abstand ab, nicht von der absoluten Position. Das Modell kann lernen: "3 Tokens zurück" — egal wo im Text.
