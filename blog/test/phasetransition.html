🧊 Phase Transitions in Neural Network Training

Neural networks don't learn gradually — they undergo abrupt phase transitions where new capabilities emerge suddenly. Each manifold represents a different learned ability. As training progresses (or model size increases), manifolds "activate" one by one — each a discrete phase transition. Points scattered in space suddenly collapse onto the next manifold when the network discovers that structure.

🔴 Scattered = not yet learned 🟡 Emerging = phase transition in progress 🟢 On manifold = ability acquired
