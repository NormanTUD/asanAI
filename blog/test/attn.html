🌀 Self-Attention: Tokens as Gravitational Particles

Each token in a sentence generates a Query ("what am I looking for?"), a Key ("what do I contain?"), and a Value ("what do I contribute?"). Attention scores are computed as the dot product of Query × Key — shown here as gravitational attraction between particles. Stronger attention = stronger pull. Different attention heads learn different relationship patterns.
