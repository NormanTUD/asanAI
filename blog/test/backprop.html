🔵 Forward Pass

Das Netzwerk berechnet von links nach rechts: Jedes Neuron nimmt seinen Input, multipliziert ihn mit dem Gewicht, addiert den Bias, und wendet die Sigmoid-Funktion an. Am Ende kommt eine Vorhersage raus.

🔴 Gradient (Backprop)

Der Gradient sagt jedem Gewicht: "Wenn ich dich ein kleines bisschen erhöhe, wird der Loss größer oder kleiner?" Das ist die Ableitung. Positiver Gradient = Gewicht muss runter. Negativer Gradient = Gewicht muss rauf.

🟢 Update

Die Gewichte werden entgegen der Gradientenrichtung angepasst: neues_gewicht = altes - lernrate × gradient. So "rutscht" das Netzwerk den Berg hinunter zum Minimum des Fehlers.
