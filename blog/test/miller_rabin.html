🔍 Miller-Rabin Primzahltest

Interaktiv erklärt – mit geometrischer Visualisierung zum Nachrechnen

📖 Was macht dieser Test?

Ziel: Herausfinden, ob eine Zahl n eine Primzahl ist (nur durch 1 und sich selbst teilbar).

Idee: Primzahlen haben besondere mathematische Eigenschaften. Wenn eine Zahl diese Eigenschaften verletzt, ist sie definitiv keine Primzahl. Wenn sie sie erfüllt, ist sie wahrscheinlich eine Primzahl.

Analogie: Stell dir einen Kreis mit n−1 Positionen vor. Wenn du von einer Startposition aus immer wieder quadrierst (Position × Position mod n), landest du bei einer Primzahl immer auf vorhersagbare Weise bei 1. Bei zusammengesetzten Zahlen passieren "unmögliche" Sprünge.

⚙️ Schritt 1: Zerlegung von n−1

Wir schreiben n − 1 = 2s · d, wobei d ungerade ist.

Das heißt: Wir teilen n−1 so oft wie möglich durch 2, bis eine ungerade Zahl übrig bleibt.

🎯 Schritt 2: Basis (Zeuge) wählen

Wir wählen eine Zahl a (zwischen 2 und n−2) als "Zeugen". Dieser Zeuge wird die Zahl "befragen".

Verschiedene Zeugen können unterschiedliche Ergebnisse liefern – je mehr Zeugen bestätigen, desto sicherer ist das Ergebnis.

📐 Schritt 3: Berechnung & Geometrische Visualisierung

Geometrische Idee: Stell dir die Zahlen 0 bis n−1 auf einem Kreis vor (wie eine Uhr mit n−1 Positionen). Die Potenzierung "springt" auf dem Kreis. Bei Primzahlen folgen die Sprünge einem regelmäßigen Muster.

Regel für Primzahlen:

1️⃣ Entweder ad mod n = 1 (wir starten direkt bei 1)

2️⃣ Oder eines der a2r·d mod n = n−1 (wir passieren −1 auf dem Weg)

Wenn keines davon zutrifft → n ist definitiv zusammengesetzt!

🔄 Mehrfach-Test (Sicherheit erhöhen)

Ein einzelner Zeuge kann sich "irren" (max. 25% Fehlerwahrscheinlichkeit). Mit k Zeugen sinkt die Fehlerwahrscheinlichkeit auf (1/4)k.

✍️ Zum Nachrechnen per Hand

Beispiel: Ist 221 eine Primzahl?

1. n−1 = 220 = 2² · 55 → s=2, d=55

2. Wähle a=174
