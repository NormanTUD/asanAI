Simplicial Complexes & The Attention Mechanism

A topological lens on how transformers see relationships

1. What is a Simplicial Complex?

A simplicial complex is a combinatorial structure built from simple pieces called simplices. A 0-simplex is a point (vertex), a 1-simplex is an edge, a 2-simplex is a filled triangle, and so on. The key rule: any face of a simplex in the complex must also be in the complex. This captures multi-way relationships — not just pairwise connections, but higher-order interactions.

2. The Attention Mechanism as a Weighted Graph

Standard self-attention computes pairwise compatibility scores between all tokens. Given queries Q, keys K, and values V:

Attention(Q, K, V) = softmax(Q·Kᵀ / √d_k) · V

This creates a complete weighted graph over tokens — every token attends to every other. But the attention weights define which edges are "strong." This is already a 1-skeleton of a simplicial complex!

3. The Bridge: From Pairwise to Higher-Order

Here's the key insight: attention weights define a filtered simplicial complex. When multiple tokens mutually attend to each other strongly, they form a clique — and cliques correspond to higher-dimensional simplices. A triangle of mutually-attending tokens is a 2-simplex. A group of four is a 3-simplex (tetrahedron).

💡 Key Insight: Persistent Homology of Attention

By varying the threshold ε from 1→0, we get a filtration — a nested sequence of simplicial complexes. Persistent homology tracks which topological features (connected components, loops, voids) appear and disappear. Features that persist across many thresholds represent robust multi-token relationships that the attention mechanism has learned. This reveals structure invisible to pairwise analysis alone.

4. Multi-Head Attention = Multiple Simplicial Complexes

Each attention head learns different relationships. Topologically, each head defines a different simplicial complex over the same vertex set (tokens). The heads capture different "aspects" of structure — some might create complexes with many connected components (local attention), others with rich higher-dimensional structure (global semantic groupings).

Simplicial Complex Vocabulary

  • k-simplex: A set of (k+1) mutually related elements
  • Face: A sub-simplex (subset of a relationship)
  • Star: All simplices containing a given vertex
  • Betti numbers: Count of k-dimensional "holes"
  • Chain complex: Algebraic structure encoding boundaries
  • Filtration: Nested sequence of complexes

Attention Mechanism Vocabulary

  • k-simplex ↔ Group of (k+1) mutually-attending tokens
  • Face ↔ Sub-group still attending to each other
  • Star ↔ All groups a token participates in
  • Betti numbers ↔ Structural "holes" in attention patterns
  • Chain complex ↔ How information flows across scales
  • Filtration ↔ Attention at varying confidence thresholds

5. Why This Matters: Topological Interpretability

Understanding attention through simplicial complexes gives us:

🔬 Research Directions

1. Topological regularization: Penalize attention patterns with undesirable topological features (e.g., too many disconnected components).
2. Simplicial attention: Directly compute attention over simplices (not just pairs) — this is simplicial transformers.
3. Persistent diagrams as features: Use persistence diagrams of attention as input to downstream tasks for interpretability.
4. Sheaves on attention complexes: Assign vector spaces (value vectors) to simplices, recovering attention as a sheaf-theoretic operation.
