What is a Monoid?

A monoid is a set of things with a way to combine any two of them (that's associative) and a special "do-nothing" identity element. Drag elements below to see it in action!

① Closure

Combining two elements always produces another element in the same set.

② Associativity

(a ∘ b) ∘ c = a ∘ (b ∘ c) — grouping doesn't matter.

③ Identity

There's a special element e where a ∘ e = e ∘ a = a.
