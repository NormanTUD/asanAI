Vietoris-Rips Complex

The Vietoris-Rips complex is one of the most fundamental constructions in TDA. Given a set of points and a radius parameter ε, we connect any two points whose distance is ≤ 2ε, and fill in all cliques (complete subgraphs) as simplices.

How It Works

  • ε = 0: Only isolated points (0-simplices).
  • Small ε: Edges appear between nearby points (1-simplices).
  • Larger ε: Triangles (2-simplices) fill in where 3 mutual edges exist.
  • Very large ε: Everything merges into one giant simplex.

Drag the slider to change ε and watch the complex evolve. Topological features (loops, connected components) appear and disappear.

Mapper Algorithm

The Mapper algorithm (Singh, Mémoli, Carlsson 2007) creates a simplified graph that captures the shape of high-dimensional data. It's like a topological "skeleton" of your dataset.

Steps

  • 1. Filter Function: Project data onto a lower-dimensional value (e.g., x-coordinate, density, eccentricity).
  • 2. Cover: Create overlapping intervals along the filter range.
  • 3. Cluster: Within each interval's preimage, cluster the points.
  • 4. Build Graph: Nodes = clusters, edges = shared points between overlapping clusters.

The result is a graph that reveals loops, branches, and flares in your data topology.

Alpha Complex (Delaunay-based)

The Alpha complex is a subcomplex of the Delaunay triangulation. It's often preferred over Vietoris-Rips because it's geometrically more faithful and computationally efficient in low dimensions.

Construction

  • Compute the Delaunay triangulation of the point set.
  • For a given radius α, keep only simplices whose circumradius ≤ α.
  • As α grows from 0 → ∞, simplices appear, tracking topological changes.

Persistence Landscapes

Persistence Landscapes (Bubenik 2015) convert persistence diagrams into a sequence of piecewise-linear functions, making them suitable for statistical analysis and machine learning.

How It Works

  • Each persistence interval [b, d] becomes a "tent" function: rises from 0 at b, peaks at (d-b)/2 at the midpoint, and falls back to 0 at d.
  • λ₁(t): The pointwise maximum of all tent functions (1st landscape).
  • λ₂(t): The pointwise 2nd-largest value (2nd landscape), etc.
  • These form a Hilbert space → you can compute means, variances, distances.

Euler Characteristic Curve

The Euler Characteristic Curve (ECC) tracks how the Euler characteristic χ = V - E + F changes as a filtration parameter grows. It's a simple, powerful topological summary.

Intuition

  • χ = (# vertices) − (# edges) + (# triangles) − ...
  • For a single connected component with no holes: χ = 1.
  • A loop decreases χ by 1; filling the loop increases it by 1.
  • The ECC plots χ(ε) as ε increases — a compact topological signature.

Reeb Graph

The Reeb graph captures the topology of a space with respect to a real-valued function. It's the theoretical foundation behind the Mapper algorithm.

Definition

  • Given a space X and a function f: X → ℝ, the Reeb graph contracts each connected component of every level set f⁻¹(t) to a single point.
  • Nodes appear where the topology of level sets changes (components merge, split, appear, or disappear).
  • The result is a graph that encodes the "skeleton" of the shape relative to f.
