Homotopy Type Theory

An interactive visual journey through the ideas that unify topology, logic, and type theory — no formulas, just shapes and intuition.

Types are Spaces, Terms are Points

In traditional programming, a type like Bool is just a label. In HoTT, every type is imagined as a space, and every value (term) of that type is a point in that space. Click on a space below to add points — each point is a concrete inhabitant of that type.

Bool has exactly 2 points: true and false. It's a discrete space — no paths connect them, because true ≠ false.

Equality is a Path

In HoTT, a proof that a = b is not just a stamp of approval — it's a path from point a to point b in the space. Different proofs of the same equality are different paths. Drag the slider to morph between two paths connecting the same endpoints.

1.5
Each colored curve is a different proof that a = b. In classical math, equality is binary. In HoTT, the space of equalities between two points can itself be rich and interesting.

Homotopy: Continuously Deforming Paths

A homotopy is a continuous deformation of one path into another. Use the slider to smoothly morph the red path into the blue path — every intermediate state is also a valid path. This is the heart of "homotopy" in HoTT.

0.00
If two paths can be continuously deformed into each other, they represent the same equality at a higher level. The deformation itself is a "path between paths" — an equality between equalities.

Higher Paths: The Tower of Equalities

Points → paths between points → homotopies between paths → homotopies between homotopies… it goes on forever. This "tower" is what makes types into ∞-groupoids. Explore different levels of this tower on a sphere.

Level 0 — Points: These are the elements (terms) of the type. On a sphere, every point on the surface is a term of type S².

The Univalence Axiom

Univalence says: if two types are equivalent (there's a perfect two-way mapping between them), then they are equal. Drag the slider to see two equivalent spaces morph into each other — they're the same type.

0.00
A triangle and a circle are topologically equivalent (homeomorphic). Univalence says we can treat them as literally the same type. This is revolutionary: structure-preserving equivalence is identity.

Why Does HoTT Matter?

HoTT provides a single language where topology, logic, and computation merge. Below is an interactive map showing how concepts from three different fields become one and the same thing in HoTT. Hover over nodes to see the connections.

HoTT lets mathematicians reason about spaces without point-set topology, lets programmers write proofs that are programs, and lets logicians work with equality that has structure. It's a unifying foundation for all of mathematics, and it's machine-checkable.
