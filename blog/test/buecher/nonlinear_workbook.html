1. The Logistic Map

The logistic map is the most fundamental example of how simple nonlinear maps can produce chaos. It is a one-dimensional map:

Starting from an initial value x₀ ∈ [0,1], iterating produces a trajectory. For r = 4, the map is fully chaotic with Liapunov exponent λ = ln 2.

2. Bifurcation Diagram

As the parameter r increases from 1 to 4, the logistic map undergoes a period-doubling cascade leading to chaos. The Feigenbaum constant δ ≈ 4.669 characterizes the rate at which bifurcations accumulate.

For r ∈ [2,3): stable fixed point. At r = 3: first bifurcation. Period doubling: 2→4→8→...→chaos at r∞ ≈ 3.5699...

3. Liapunov Exponent

The Liapunov exponent quantifies sensitive dependence on initial conditions — the hallmark of chaos. For the logistic map:

λ > 0: chaotic (nearby trajectories diverge exponentially). λ < 0: periodic/stable. λ = 0: bifurcation point.

4. Hénon Map (2D Chaos)

The Hénon map extends chaos to two dimensions. It is the most studied 2D map with a strange attractor:

For a = 1.4, b = 0.3: strange attractor with fractal dimension ≈ 1.26. The map is invertible (b ≠ 0) and dissipative (|b| < 1).

Strange attractor — fractal structure visible at all scales

5. Lorenz System (Continuous Chaos)

The Lorenz system is the most famous dissipative system with chaotic behavior, arising from fluid convection:

For σ=10, r=28, b=8/3: the butterfly attractor. Sensitive dependence on initial conditions was first discovered here by Edward Lorenz in 1963.

6. Fractals

Fractals are self-similar structures with non-integer dimension. Key examples include the Mandelbrot set, Julia sets, and iterated function systems (IFS).

The Mandelbrot set M = {c ∈ ℂ : the orbit of 0 under z → z² + c does not escape to ∞}. Click on the canvas to zoom in!

7. Cellular Automata

Cellular automata are discrete dynamical systems on a lattice. Each cell updates based on its neighbors according to a fixed rule.

For 1D elementary CA with k=2, r=1: 256 possible rules. Rule 90 (XOR of neighbors) produces the Sierpinski triangle. Rule 30 produces apparent randomness.

Rule 90: a(t+1) = a(i-1,t) XOR a(i+1,t)
